# Book API
RESTful web API using node and express

## Database
start mongodb

## Run API
gulp

## Notes
For Homebrew mongodb, use `brew services start mongodb`

Add data by POSTing 
```
{
    "author": "Frank Herbert",
    "title": "Dune",
    "genre": "Science Fiction"
}
```
and similar to http://localhost:8000/api/books as raw JSON
